# LogiMind-ai-agent
transportation management automation agent

Activate python environment first - 
venv\scripts\activate

backend run command : 
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

docker activation : 
docker compose up
docker-compose up -d --build 

frontend run command : 
npm run dev

run a mobile app :
npx expo start -c
